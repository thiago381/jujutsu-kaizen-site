Jujutsu Kaizen Site

Projeto front-end inspirado no universo de Jujutsu Kaisen, desenvolvido com HTML, CSS e JavaScript puro.

Este projeto está sendo utilizado como laboratório prático para evolução em desenvolvimento web, organização de código e versionamento profissional com Git.

Objetivo

Construir uma fanpage completa enquanto pratico:

Estruturação semântica em HTML

Organização de layout com CSS

Interações com JavaScript

Versionamento com commits organizados

Planejamento de funcionalidades via Issues

Tecnologias Utilizadas

HTML5
CSS3
JavaScript

Funcionalidades Implementadas

Estrutura base do site
Topbar com navegação
Menu lateral (hambúrguer) com JavaScript
Modal de login centralizado
Páginas internas interligadas:

Personagens

Episódios

Layout responsivo
